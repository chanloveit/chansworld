# [[BOJ/백준 14786 - Ax+Bsin(x)=C②]](https://www.acmicpc.net/problem/14786)
![백준 14786](/images/Boj14786.jpg)

## 문제 이해
A, B, C가 주어지고 Ax + Bsin(x) = C를 만족하는 x를 찾는 문제다.

## 핵심 아이디어
x에 대하여 식을 정리하면 x = (C / A) - (Bsin(x) / A)로 나타낼 수 있다.  
A, B, C는 모두 자연수이므로 x의 범위는 0 < x < (C / A) + (B / A)로 정리할 수 있다.  
f(x) = Ax + Bsin(x)는 거의 선형이므로 이분 탐색으로 근사값을 찾을 수 있다.  
반복문을 더 크게 할수록 더 정확한 값이 나오지만 100번 정도만 돌려도 정답을 받을 수 있다.

## 소스코드
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>
#include <iomanip>
using namespace std;

int a, b, c;

double f(double x){
	return a * x + b * sin(x);
}

int main() {
	ios_base::sync_with_stdio(false);
	cin.tie(NULL);
	
	cin >> a >> b >> c;

	double left = 0;
	double right = (double)c / a + (double)b / a;
	for(int i = 0; i < 100; i++){
		double mid = (left + right) / 2.0d;
		if(f(mid) < c){
			left = mid;
		}

		else{
			right = mid;
		}
	}

	cout << fixed << setprecision(9) << (left + right) / 2.0 << '\n';
	return 0;
}
}
```
## 풀이
이분 탐색으로 left와 right의 중간값 mid를 구하고, f(mid) < c이면 left를 mid로, 아니면 right를 mid로 좁혀나간다.  
`iomanip`의 `setprecision(9)`으로 소수점 아래 9번째 자리까지 출력한다.
