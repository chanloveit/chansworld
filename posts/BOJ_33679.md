# [[백준 33679 - 세기의 대결]](https://www.acmicpc.net/problem/33679)
![백준 33679 - 세기의 대결](/images/BOJ_33679/Boj33679.jpg)

## 문제 이해
두 사람 YJ와 HG가 각자 배열을 가지고 있고, 각 배열에서 원형 LIS(최장 증가 부분 수열)의 길이를 계산하여 더 긴 쪽이 승리하는 문제. 같으면 둘 다 승리.

## 핵심 아이디어
주어진 배열에서 시작점이 i = 0으로 정해지지 않았으므로, 원형 LIS를 계산하기 위해 배열을 두 번 복사하여 선형으로 만들고, 각 시작점에서 LIS를 계산한다. 

## 소스코드
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
int circular_lis(vector<int>& arr){
    int n = arr.size();

    vector<int> v = arr;
    v.insert(v.end(), arr.begin(), arr.end());
    int res = 0;

    for(int start = 0; start < n; start++){
        vector<int> lis;

        for(int i = start; i < start + n; i++){
            auto it = lower_bound(lis.begin(), lis.end(), v[i]);
            if(it == lis.end()) {
                lis.push_back(v[i]);
            }
            else{
                *it = v[i];
            } 
        }

        res = max(res, (int)lis.size());
    }

    return res;
}


int main(void) {
    ios::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;
    vector<int> a(n);
    vector<int> b(n);

    for(int i = 0; i < n; i++){
        cin >> a[i];
    }
    for(int i = 0; i < n; i++){
        cin >> b[i];
    }

    int yj = circular_lis(a);
    int hg = circular_lis(b);

    if(yj > hg) cout << "YJ Win!";
    else if(yj < hg) cout << "HG Win!";
    else cout << "Both Win!";
}
```

## 풀이
일반적인 LIS(Longest Increasing Subsequence)는 배열의 시작점이 고정되어 있다.
하지만 이 문제에서는 배열이 원형이기 때문에, 시작점이 어디인지에 따라 LIS의 길이가 달라질 수 있다.
예를 들어 배열이 다음과 같다고 하자.   
`
3 1 2
`   
선형 배열 기준 LIS는 1 2로 길이가 2이지만 원형으로 보면    
`
1 2 3
`   
이라는 수열을 만들 수 있어 LIS 길이가 3이 된다.  
따라서 원형 LIS를 구하려면 모든 시작점에 대해 LIS를 계산해야 한다.   
이를 위해 배열을 다음과 같이 두 번 이어 붙인다.   
`
arr = [a1, a2, ..., an]
v  = [a1, a2, ..., an, a1, a2, ..., an]
`    
이렇게 하면 원형 배열에서 시작점을 옮기는 문제를 선형 배열의 구간 문제로 바꿀 수 있다.   
이후 배열을 n의 크기로 슬라이싱하여 각 구간에 대한 LIS의 길이 중 가장 큰 것을 비교한다.
